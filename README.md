## Deskripsi Aplikasi
**Guessasaur** adalah aplikasi kuis bertemakan dinosaurus yang berisi pertanyaan seputar nama-nama dinosaurus. 
Aplikasi ini berisi tebak tebakan nama dinosaurus berdasarkan gambar yang ditampilkan.
